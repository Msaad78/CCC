polyline

  • The polyline element connects multiple points together with straight lines.
  • Points are defined using a string.
  • Each point is written as a pair of coordinates x,y and separated by a space.
  • The line is filled by default, but the path is not closed, so to create a complete outline it is necessary make the first and last points the same.
  • Notice that in the last polyline the top point is not nicely mitred. This is because the stroke path is not closed.
