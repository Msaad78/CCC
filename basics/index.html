Shapes

SVG images are composed of elements (similar to HTML).

The language is rich and expressive, but a quick start is easy using basic shapes (rect, circ and paths).

Rectangles

  • Rectangles can be created using the rect element.
  • rect takes four parameters: an x & y position, a width and height.
  • Shapes are filled with black by default, and have no outline.
  • The coordinate system starts in the top left of the image (where x=0 and y=0)
  • The colour of any shape can be defined using the fill attribute which can be set to any HTML-like colour.
  • Shapes are drawn in the order they appear in the SVG element.
  • The outline colour of a shape can be defined using the stroke attribute.
  • The thickness of the outline si controlled using the stroke-width attribute.
  • Shapes may also have a none fill.

Circles

  • Circles can be created using the circ element.
  • circ takes three parameters: a centre cx & cy denoting the circle's centre, and r rescribing its radius.
  • Circles can be filled and outlined.

Ellipses

  • Ellipses are like circles but they contain two radii (rx & ry).

Lines

There are several ways to draw lines in SVG.

line

  • The line element draws a straight line between two points.
  • Points are defined using four parameters x1, y1, x2 & y2
  • Note that if no stroke colour is defined, the line will not be visible.

polyline

  • The polyline element connects multiple points together with straight lines.
  • Points are defined using a string.
  • Each point is written as a pair of coordinates x,y and separated by a space.
  • The line is filled by default, but the path is not closed, so to create a complete outline it is necessary make the first and last points the same.

polygon

  • The polygon element connects multiple points together, just like polyline, but, it automatically closes the path without duplicating the start point.

Paths

  • The path element is the most flexible drawing tool in SVG.
  • path element works similarly to polyline insofar as a series of points can be defined in a string.
  • In addition to defiing points it's also possible to define how the points are connected together (with straight lines or curves) or not connected at all (by moving).
