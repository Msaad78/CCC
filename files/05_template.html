  • This template contains three lines.
  • The first line declares that it's an XML document.
  • The second line includes the SVG tag, which must include the SVG namespace specifier (this is not necessary when embedding SVG into XML).
  • The last line closes the SVG element.
  • Note that in SVG, all attributes bust be quoted (SVG is more strict than HTML5 in this respect because it is a dialect of XML).
